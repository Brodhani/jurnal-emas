# Jurnal Emas Pro — XAUUSD (PWA)

Jurnal trading XAUUSD lengkap: kalkulator multi-layer, auto-stop harian, checklist disiplin,
berita gold live, kalender news + pengingat, keuangan pribadi IDR, converter USD-IDR,
to-do trading, dan AI coach.

## Cara pasang di GitHub Pages

1. Buka **github.com** → login → **New repository** → nama bebas (mis. `jurnal-emas`) → Public → Create.
2. Klik **Add file → Upload files** → upload SEMUA file di folder ini
   (`index.html`, `manifest.json`, `sw.js`, `icon-192.png`, `icon-512.png`) → Commit.
3. Buka **Settings → Pages** → Source: **Deploy from a branch** → Branch: **main** / root → Save.
4. Tunggu 1-2 menit → buka URL: `https://USERNAME.github.io/jurnal-emas/`

## Pasang di iPhone

1. Buka URL di **Safari**.
2. Tap tombol **Share** (kotak panah ke atas) → **Add to Home Screen** → Add.
3. Aplikasi muncul di home screen seperti app biasa (fullscreen, ada ikonnya).

## Fitur AI (opsional)

Harga gold & kurs USD-IDR sudah live gratis tanpa setting apa pun.
Untuk berita AI, bias harian, kalender news + pengingat, analisa scalping/intraday, dan AI coach:

1. Buat API key di **console.anthropic.com** → API Keys (berbayar sesuai pemakaian, sangat murah untuk pemakaian pribadi).
2. Di aplikasi: tab **Alat → Koneksi AI** → tempel key → Simpan.

Key hanya tersimpan di HP kamu (localStorage) dan dikirim langsung ke Anthropic.
JANGAN pernah taruh API key di dalam kode / repo GitHub.

## Catatan

- Data tersimpan di perangkat (localStorage) — tidak sinkron antar-device.
- Ekspor CSV rutin dari tab Jurnal sebagai backup.
